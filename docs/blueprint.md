# **App Name**: MediConnect Hub

## Core Features:

- User Authentication & Roles: Secure sign-in for Doctor and Admin users, with distinct access privileges and interface views based on assigned roles.
- AI-powered Letter Draft Assistant: A generative AI tool that assists doctors in drafting consultation letters, suggesting content based on patient details and selected templates.
- Consultation Letter Management: Create, edit, save, and manage consultation letters, with functionality to import (PDF/DOCX) and export (PDF/Image) letters for external use.
- Secure Intra-System Sharing: Facilitate secure sharing of consultation letters directly with other registered doctors within the system, including tracking of sharing status.
- Patient Profile Linking: Maintain essential patient profiles, linking each profile to its associated consultation letters for comprehensive record keeping.
- Inbox & Notifications: Provide doctors with an inbox to view received shared letters and a real-time notification system for new incoming shares.
- Global Search & Filtering: Implement robust search and filter capabilities to quickly locate specific patients, letters, or doctors across the platform.

## Style Guidelines:

- Primary color: A deep, authoritative Royal Blue (#3D29A3), selected to convey trust, professionalism, and medical authority, fitting for a clinical context.
- Background color: A very subtle, cool-toned gray-blue (#F1F0F4), providing a clean, clear, and professional backdrop for content, promoting focus and readability in a light color scheme.
- Accent color: A brighter, dynamic Cerulean Blue (#5282E0), chosen for its analogous relationship to the primary hue, while offering strong contrast for interactive elements, highlights, and calls to action.
- Headlines font: 'Playfair', a modern serif typeface, provides elegance and sophistication suitable for official document titles and important headers, offering a high-end feel.
- Body text font: 'PT Sans', a clear and warm humanist sans-serif, ensuring optimal readability and a modern aesthetic for detailed medical information and user interface text.
- Utilize a consistent set of minimalist line-art or glyph-based icons to maintain a clean and modern professional appearance, prioritizing clarity and immediate recognition for all interface actions and concepts.
- Implement a structured, clean, and intuitive layout with clear visual hierarchy, utilizing ample white space to enhance readability and reduce cognitive load for busy medical professionals.
- Employ subtle, purposeful animations for transitions between sections, confirmation of actions (e.g., sharing a letter, saving), and feedback, ensuring a smooth and responsive user experience without being distracting.
